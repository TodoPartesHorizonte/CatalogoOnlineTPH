## 2. DEFAULT ARCHITECTURE & CONVENTIONS

* **ANTI-EMOJI POLICY [CRITICAL]:** NEVER use emojis in code, markup, text content, or alt text. Emojis are BANNED.
* **Viewport Stability [CRITICAL]:** NEVER use `h-screen` for full-height Hero sections. ALWAYS use `min-h-[100dvh]` to prevent catastrophic layout jumping on mobile browsers.
* **Hardware Acceleration:** Use `translate3d(0,0,0)` or `will-change` for complex animations.

* Avoid generic increments. Use a 4px or 8px grid but allow for optical corrections.

**Rule 2: Color Calibration**
* **THE LILA BAN:** The "AI Purple/Blue" aesthetic is strictly BANNED. No purple button glows, no neon gradients. Use curated, professional palettes.

**Rule 3: Typography**
